# Padre Gino's

A React pizza ordering app built while working through **Complete Intro to React** on Frontend Masters.

## Live Project

Frontend: https://illustrious-peony-ab8c74.netlify.app/

Backend API: https://pizza-server-app-3uqk.onrender.com/

## Overview

Padre Gino's is a client-side React application for browsing pizzas, adding items to a cart, submitting orders, viewing past orders, and sending contact form submissions.

## React Concepts Practiced

This project covers a wide range of modern React concepts, including:

- component-based architecture

- props and JSX

- state with `useState`

- side effects with `useEffect`

- custom hooks

- React Context for shared state

- controlled and uncontrolled forms

- React 19 form actions

- portals

- error boundaries

- routing with TanStack Router

- server-state management with TanStack Query

- testing with Vitest and React Testing Library

- environment-based API configuration for development and production

## Deployment

The client app is deployed on Netlify, and the API is deployed separately on Render.
